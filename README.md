## Instructions

To get started please follow the instructions bellow:

* git clone https://github.com/KWagner91/Recipe-Shopper
* cd Recipe-Shopper

Install dependencies and run with:

_npm_

> npm install
> npm run serve


and

_Angular_

> npm install -g @angular/cli
> ng serve --open

And then visit localhost:4200
